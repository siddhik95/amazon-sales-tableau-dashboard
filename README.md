# Amazon Sales Dashboard | Tableau Project

## Project Overview

This project analyzes Amazon sales data using Tableau to uncover key business insights related to sales performance, profitability, product categories, customer behavior, and return trends.

The dashboard is designed to help stakeholders monitor business performance and make data-driven decisions.

---

## Objectives

- Analyze overall sales performance
- Identify top-performing product categories
- Track profit trends across regions
- Monitor return rates
- Understand customer purchasing patterns
- Generate actionable business insights

---

## Tools Used

- Tableau
- Microsoft Excel
- Data Cleaning
- Data Visualization

---

## Key Performance Indicators (KPIs)

- Total Sales
- Total Profit
- Total Orders
- Return Rate
- Average Order Value

---

## Dashboard Features

### Sales Performance Dashboard
- Sales by Category
- Sales by Sub-Category
- Sales Trend Analysis
- Regional Sales Distribution
- Top Products Analysis

### Profit & Returns Dashboard
- Profit by Category
- Profit by Region
- Return Rate Analysis
- Customer Segment Analysis

---

## Key Insights

- Technology category generated the highest sales.
- Certain regions contributed significantly to overall profit.
- Return rates varied across product categories.
- A small number of products contributed a large portion of revenue.
- Sales showed seasonal fluctuations throughout the year.
